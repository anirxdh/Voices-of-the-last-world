# Hack #5 Submission Guide

## What Judges Score

According to the official rules, submissions are judged on:

- creativity and originality: 40%
- effective use of partner technology and ElevenLabs tools: 40%
- quality of presentation and demo: 20%

Sources:
- https://hacks.elevenlabs.io/hackathons/4
- https://hacks.elevenlabs.io/terms

## What This Means For This Repo

To place well, this project has to look like:

1. a creative cinematic product
2. a clear ElevenLabs-powered experience
3. a clear Kiro spec-driven workflow

If the submission only shows the web app, it leaves points on the table.

## Strong Submission Positioning

Use this wording:

`Voices of the Last World is a cinematic crisis-simulation game built with Kiro's spec-driven development workflow and ElevenLabs voice generation. Players assemble two Archive minds, watch them debate a global crisis, and then commit the final response.`

## Demo Structure

### 1. Hook

Show the polished intro and one strong mission moment immediately.

Goal:
- make it feel premium in the first few seconds

### 2. Show Kiro

Show:
- the spec file
- Kiro reading the spec
- Kiro applying the change
- the changed app behavior

Goal:
- prove spec-driven development, not just mention it

### 3. Show ElevenLabs

Show:
- distinct character voices
- the Archive onboarding voice
- result narration

Goal:
- make ElevenLabs usage impossible to miss

### 4. Show The Game Loop

Show:
- scenario selection
- choosing two agents
- agent debate
- operator response choice
- final outcome

## Best Features To Highlight

- two-agent debate loop
- differentiated voice identity by agent
- cinematic presentation and mission framing
- operator choice after autonomous debate
- result narration and consequence screen

## Weak Submission Patterns To Avoid

- talking about Kiro without showing it
- showing only the final site
- letting the app fall back to browser speech in the recorded demo
- cluttered UI or obviously unfinished code structure
- demoing too many features without one memorable through-line
